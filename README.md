# v2ray_user_manager
基于V2Ray + VMess + WS + TLS + Nginx 服务端的用户管理脚本

**目前只适合 Debian/Ubuntu 家族**

## 非root用户使用方式：
`curl -L https://raw.githubusercontent.com/whnan/v2ray_user_manager/master/v2ray_user_manager.sh | sudo bash`

## root用户使用方式：
`curl -L https://raw.githubusercontent.com/whnan/v2ray_user_manager/master/v2ray_user_manager.sh | bash`
